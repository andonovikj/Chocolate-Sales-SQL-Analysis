# Chocolate-Sales-SQL-Analysis

*Project Overview*

This project analyzes a chocolate company's sales performance using PostgreSQL. The dataset contains over 3,000 sales transactions from multiple countries between 2022 and 2024. The analysis focuses on identifying revenue trends, top-performing products, country performance, salesperson effectiveness, and year-over-year business growth.

Skills Demonstrated:
- PostgreSQL
- Data Cleaning
- Aggregate Functions
- GROUP BY
- ORDER BY
- Common Table Expressions (CTEs)
- Window Functions (RANK, LAG)
- Date Functions
- Business KPI Analysis
- Pareto (80/20) Analysis
- Sales Trend Analysis
