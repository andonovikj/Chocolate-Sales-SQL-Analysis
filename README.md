# Chocolate Sales SQL Analysis

*Project Overview*

This project analyzes a chocolate company's sales performance using PostgreSQL. It's based on the **Chocolate Sales** dataset from Kaggle, which contains sales transactions from 2022–2024 across multiple countries and products.

Source: https://www.kaggle.com/datasets/saidaminsaidaxmadov/chocolate-sales

The analysis focuses on identifying revenue trends, top-performing products, country performance, salesperson effectiveness, and year-over-year business growth.

**Skills Demonstrated:**
- PostgreSQL
- Data Cleaning
- Aggregate Functions
- GROUP BY
- ORDER BY
- Common Table Expressions (CTEs)
- Window Functions (RANK, LAG)
- Date Functions
- Business KPI Analysis
- Pareto (80/20) Analysis
- Sales Trend Analysis

To complement the SQL analysis, I created an **interactive Power BI dashboard** that visualizes key business insights, including revenue trends, product and country performance, salesperson performance, and overall sales KPIs. Interactive slicers allow users to explore the data dynamically across multiple dimensions.
